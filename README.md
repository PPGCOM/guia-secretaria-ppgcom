# Guia de Processos - Secretaria MP-PPGCOM

Um site informativo e organizado para consulta de processos da secretaria do Programa de Pós-Graduação em Comunicação Social (MP-PPGCOM) da PUCRS.

## 📋 Sobre o Projeto

Este site consolida e detalha os procedimentos da Secretaria do MP-PPGCOM, integrando:
- Informações sobre modalidades de alunos (regulares, especiais, convênios, G-PG, G+1)
- Processos relacionados a disciplinas
- Procedimentos de bancas (qualificação e defesa)
- Informações sobre bolsas
- Outros processos administrativos
- Informações oficiais do programa PUCRS

## 🚀 Como Usar

### Acessar o Site
O site está hospedado em GitHub Pages e pode ser acessado através de um domínio permanente.

### Atualizar o Conteúdo

#### Opção 1: Editar Diretamente no GitHub (Recomendado para Iniciantes)
1. Acesse o repositório no GitHub
2. Clique no arquivo `index.html`
3. Clique no ícone de lápis (Edit)
4. Faça as alterações desejadas
5. Clique em "Commit changes"
6. As mudanças serão publicadas automaticamente em poucos minutos

#### Opção 2: Usar Git (Para Usuários Avançados)
```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/guia-secretaria-ppgcom.git
cd guia-secretaria-ppgcom

# Fazer alterações nos arquivos
# Editar index.html conforme necessário

# Adicionar e fazer commit das mudanças
git add .
git commit -m "Descrição das alterações"

# Enviar para GitHub
git push origin master
```
